---
title: Pricing and Plans
description: How GitHub structures its pricing and plans.
---

GitHub offers a free tier aimed at individuals and small teams, which includes unlimited public and private repositories along with a metered allotment of Actions minutes and storage. Above that sits a paid Team plan that adds finer-grained access controls, protected branches, and collaboration features, and an Enterprise plan that layers on advanced security, compliance, auditing, and SSO/SAML support for larger organizations.

For organizations that cannot use the SaaS product, GitHub Enterprise Server provides a self-hosted, on-premises version of GitHub with periodic syncing to GitHub Connect for some hosted features. Paid plans are generally billed per active user per month, with add-ons (such as GitHub Advanced Security or extra Actions/storage minutes) priced separately rather than bundled into a single flat fee.
