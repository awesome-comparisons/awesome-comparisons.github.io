---
title: History
description: The origins of 14 major programming languages, who created them, when, and why.
---

## JavaScript

Created by Brendan Eich in 1995 while at Netscape, reportedly designed and prototyped in about ten days. Originally called Mocha, then LiveScript, it was renamed JavaScript for marketing reasons tied to the popularity of Java at the time. It was later standardized as ECMAScript.

## TypeScript

Developed by Microsoft, with Anders Hejlsberg as lead architect, and first released publicly in 2012. It was created to bring static typing and better tooling to large JavaScript codebases as web applications grew in scale and complexity.

## Python

Created by Guido van Rossum and first released in 1991 as a successor to the ABC language. The name comes from Monty Python's Flying Circus, reflecting van Rossum's desire for a fun, readable language.

## Go

Designed at Google starting in 2007 by Robert Griesemer, Rob Pike, and Ken Thompson, and released publicly in 2009. It grew out of frustration with slow builds and complexity in the large C++ services Google maintained.

## Rust

Begun as a personal project by Graydon Hoare in 2006 and later sponsored by Mozilla starting around 2009. It reached its 1.0 stable release in 2015, aimed at systems programming with guaranteed memory safety.

## Java

Created by James Gosling and colleagues at Sun Microsystems, released in 1995. It began in 1991 as a project called Oak intended for interactive television and set-top boxes before being retargeted at the emerging web.

## Kotlin

Developed by JetBrains and first unveiled in 2011, reaching a stable 1.0 release in 2016. It is named after Kotlin Island near Saint Petersburg, echoing Java's naming after an island as well.

## Swift

Introduced by Apple in 2014, led primarily by Chris Lattner, as a modern replacement for Objective-C on Apple platforms. It was open-sourced in 2015 to broaden its use beyond Apple's ecosystem.

## C

Developed by Dennis Ritchie at Bell Labs between 1969 and 1973, growing out of the earlier B language. It was created largely to rewrite the Unix operating system in a portable, higher-level language than assembly.

## C++

Created by Bjarne Stroustrup at Bell Labs, starting in 1979 as "C with Classes" and released under the C++ name in 1985. It extended C with object-oriented and later generic programming features while retaining low-level control.

## C#

Designed by Microsoft, with Anders Hejlsberg as chief architect, and released in 2000 alongside the .NET Framework. It was created to give .NET a modern, type-safe managed language comparable to Java.

## Ruby

Created by Yukihiro "Matz" Matsumoto in Japan, with first public release in 1995. It blended ideas from Perl, Smalltalk, Eiffel, and Lisp to make a language Matsumoto felt was more enjoyable to write.

## PHP

Originally written by Rasmus Lerdorf in 1994 as a set of Common Gateway Interface scripts called "Personal Home Page Tools." It evolved through contributions from others into a full scripting language for dynamic web pages.

## Elixir

Created by José Valim, a former Ruby on Rails core team member, and first released in 2011. It was built on top of the battle-tested Erlang VM to combine Ruby-like ergonomics with Erlang's concurrency and fault tolerance.
