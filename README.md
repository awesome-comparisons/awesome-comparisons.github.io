- [Backend Frameworks](https://awesome-comparisons.github.io/content/02-frameworks/04-backend/20260715093700_routing) - Routing, middleware, and request lifecycle of backend web frameworks including Express, Django, Spring Boot, Laravel, and Rails.
- [Game Engines](https://awesome-comparisons.github.io/content/02-frameworks/05-game-engines/20260725090000_philosophy) - Scripting languages, rendering pipelines, and licensing of game engines including Unity, Unreal Engine, Godot, and GameMaker.

## Browsing

Every entry above links to the start of that comparison. From there, the sidebar lets you browse the rest of its chapters (history, architecture, performance, ecosystem, and more) or jump to any other comparison, and `Cmd/Ctrl+K` opens search across all of them. Each page shows a reading time estimate and links to the previous/next chapter in its section.
